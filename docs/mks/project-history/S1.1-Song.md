# Sprint S1.1 — Song

**Status:** ✅ Concluída

## Objetivo

Criar o modelo `Song`, que representa uma música dentro do Melody Sync.

## Implementado

- Modelo `Song`
- Organização dos atributos
- Propriedades derivadas
- Uso de `dataclass(slots=True)`

## Decisões

- O modelo representa apenas dados.
- Não possui lógica de negócio.
- Armazena os valores originais.
- Propriedades apenas para informações derivadas.

## Resultado

O `Song` tornou-se o modelo base utilizado por todo o Core.
