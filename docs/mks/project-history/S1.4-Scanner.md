# Sprint S1.4 — Scanner

**Status:** ✅ Concluída

## Objetivo

Implementar o Scanner responsável por transformar uma biblioteca musical em uma lista de objetos `Song`.

## Implementado

- Integração com o Discovery.
- Integração com o Metadata.
- Criação do pipeline de leitura da biblioteca.
- Retorno de uma lista de objetos `Song`.

## Decisões

- O Scanner atua apenas como orquestrador.
- O Scanner não conhece a CLI.
- O Scanner não calcula estatísticas.
- O Scanner apenas coordena Discovery e Metadata.

## Resultado

O Melody Sync passou a converter uma biblioteca musical em objetos do domínio de forma organizada e desacoplada.
