# Sprint S1.5 — Statistics

**Status:** ✅ Concluída

## Objetivo

Criar o módulo responsável por gerar estatísticas da biblioteca musical.

## Implementado

- Modelo `LibraryStatistics`.
- Função `calculate_statistics()`.
- Contagem de músicas, artistas e álbuns.
- Estatísticas de tamanho, duração, bitrate e formatos.

## Decisões

- Estatísticas são calculadas separadamente do Scanner.
- O modelo armazena apenas dados.
- Conversões são feitas através de propriedades.
- O módulo não possui dependência da interface.

## Resultado

O Core passou a fornecer um resumo completo da biblioteca musical para qualquer interface.

## Próxima Sprint

Integrar o Core à interface de linha de comando.
